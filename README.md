# OSCAT Network Documentation

Dieses Repository enthält die Dokumentation für die **OSCAT Network Library**.

Die Dokumentation ist verfügbar unter: [https://oscat-network.readthedocs.io](https://oscat-network.readthedocs.io)

## Kategorien

- **NETWORK** - Netzwerkprotokolle und Kommunikation
- **NET_VAR** - Netzwerkvariablen
- **OTHER** - Sonstige Funktionen
- **TELNET_VISION** - Telnet und Visualisierung
- **WEATHER** - Wetterdienste und Datenauswertung

## Lokale Entwicklung

```bash
pip install -r docs/requirements.txt
mkdocs serve
```
