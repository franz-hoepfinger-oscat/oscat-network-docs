| | |
|:---|:---|
| **Type** | Funktionsbaustein |
| **INPUT	Iin_Y** | INT : (Y-Koordinate) |
| **Iin_X** | INT : (X-Koordinate) |
| **OUTPUT	Oby_ATTR** | BYTE : (Farbinformation an Position X/Y) |
| **IN_OUT	Xus_TN_SCREEN** | us_TN_SCREEN |
