## TN_SC_XY_ERROR

| | |
|:---|:---|
| **Type	Function** | BOOL |
| **INPUT** | X: INT: (X coordinate) |
| **Y** | INT: (Y coordinate) |
| | The module TN_SC_XY_ERROR checks whether the specified coordinate   is within the screen area. If the check fails, as result is passes TRUE. |
